# vue-example-hello

## 一、vue基础

### 1. 安装vue-devtools

​	**官方网址：**https://github.com/vuejs/vue-devtools#vue-devtools

#### 1-1. 安装步骤

​	由于公司电脑的组策略禁用Chrome安装扩展插件，因此只能用Firefox浏览器。

1. 从github下载到本地

   ```
   https://github.com/vuejs/vue-devtools.git
   ```

2. 解压后进入vue-devtools目录，进行安装

   ```shell
   cd vue-devtools
   npm install
   ```
